# AutoTap

TrollStore 环境下的 iOS 自动点击 dylib（悬浮窗 UI + 多步骤延迟 + 循环次数 + 本地方案保存/切换）。

## 功能

- 悬浮按钮：可拖拽、点击展开面板
- 自由取点：在屏幕上点一下记录坐标
- 步骤延迟：每个步骤可设置点击后等待毫秒数
- 循环次数：整套步骤循环执行 N 次
- 方案管理：本地 JSON 保存/加载/删除

## 构建

使用 Theos：

```bash
make package
```

## 使用

1. 打开目标 App（dylib 被注入后会出现悬浮按钮）
2. 进入面板 → 添加步骤（取点）→ 设置延迟
3. 设置循环次数 → 开始执行
4. 保存方案，下次可直接加载切换
